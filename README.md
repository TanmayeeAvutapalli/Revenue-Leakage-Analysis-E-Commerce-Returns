# Revenue-Leakage-Analysis-E-Commerce-Returns
Problem Statement
Product returns are one of the biggest silent revenue killers in e-commerce. Unlike cart abandonment or low conversion rates, returns happen after the sale — meaning the business has already spent on inventory, packaging, shipping and delivery, only to lose it all when the product comes back.
With a return rate of 33.55% and ₹4.26Cr in revenue leakage, this project investigates the real question every e-commerce business needs to answer:
Why are customers returning products — and what should the business do about it?
Objective
To analyze 10,000 e-commerce orders and identify return patterns across product categories, price ranges, regions, months and return reasons — and translate those findings into a data-driven action plan.

Tools Used

SQL — data extraction, return rate calculations, views
Power BI —  interactive dashboard with DAX measures
Dataset
Synthetic Indian e-commerce dataset generated using Python containing 10,000 orders across 8 product categories, 10 Indian cities, 7 return reasons and 2 years of order data (2023–2024).

Key Insights

Overall return rate is 33.55% contributing to ₹4.26Cr in total revenue leakage
Electronics has the highest return rate at 46% — nearly 1 in 2 orders returned
Wrong Item Delivered is the most frequent return reason across all categories
February and November are peak return months linked to pre-sale inventory quality gaps
Delhi, Kolkata and Chennai have elevated return rates driven by damaged deliveries and quality issues
Premium products (above ₹10,000) have the highest return rate at 45% — highest revenue impact per return
Average rating of returned orders is 3.38 vs delivered orders — confirming low product satisfaction drives returns
Business Impact
Findings from this analysis led to a data-driven action plan projected to reduce the overall return rate from 33.55% to below 20% — recovering a significant portion of the ₹4.26Cr revenue leakage through warehouse verification checks, pre-sale quality controls and regional packaging improvements.
